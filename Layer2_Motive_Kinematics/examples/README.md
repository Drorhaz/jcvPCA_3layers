# Examples

This directory holds usage notes only. **No Motive CSV sample data is committed.**

Before Milestone 1, provide one representative **mixed Motive CSV** (global bone quaternions) locally, e.g.:

```bash
layer2-motive run-stage \
  --input /path/to/your/motive_export.csv \
  --output-dir outputs/run1 \
  --stage 00
```

Stage-by-stage execution and stop-and-validate protocol: see `04_AGENT_STOP_AND_VALIDATE_PROTOCOL.md` at repo root.
